Регистрация нового читателя

Процесс:

1. Посетитель заполняет бумажную анкету, предоставляя свои личные данные (ФИО, телефон, email).
2. Библиотекарь вводит данные из анкеты в компьютер, вызывая у системы ISB ключевой метод registerAccount.
3. ISB, используя метод registerAccount, создает учетную запись (новый экземпляр объекта Account).
   Конструктор класса Account инициализирует объект переданными данными (ФИО, телефон, email) и автоматически присваивает начальные значения:
    a. Генерируется уникальный идентификатор (id),
    b. Параметр status устанавливается в значение "Активен",
    c. Список книг на руках (BooksOnHand) инициализируется как пустой (List<String>),
    d. Счетчик текущих задолженностей (currentLoans) устанавливается в 0,
    e. Список текущих бронирований (currentReservations) инициализируется как пустой.
   Созданный объект Account добавляется в коллекцию accounts (поле Map<String, Account> класса ISB).
4. ISB возвращает Библиотекарю подтверждение об успешном завершении регистрации ("регистрация завершена").
5. Библиотекарь, получив подтверждение от системы, формирует и передает Посетителю его читательский билет, содержащий QR-код. Этот QR-код должен содержать уникальный идентификатор (id) созданной учетной записи, который в дальнейшем может быть использован для быстрой аутентификации читателя при помощи метода ISB.scanQRCode(qrData: String).
```
@startuml

actor "Visitor" as Visitor
actor "Librarian" as Librarian
participant "ISB" as ISB
participant "Account" as Account

Visitor -> Librarian : Подать бумажную анкету\n(ФИО, телефон, почта)
Librarian -> ISB : registerAccount(fullName, phone, email)
activate ISB

ISB -> Account : create Account(fullName, phone, email)
activate Account

Account --> ISB : Account created with:\n- generated ID\n- status = "Active"\n- BooksOnHand = []\n- currentLoans = 0\n- currentReservations = []

deactivate Account

ISB --> Librarian : Регистрация прошла успешно\n(id, qrData)
deactivate ISB

Librarian -> Visitor : Выдать читательский билет с QR-code\n(с id аккаунта)

@enduml
```
