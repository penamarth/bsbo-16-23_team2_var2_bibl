Регистрация нового читателя

Процесс:

1. Посетитель заполняет анкету.
2. Библиотекарь вводит данные в компьютер (ИСБ).
3. ИСБ создает учетную запись, содержащую уникальный ID. А также присваивает статус "Активен", начальные условия (пустой список одолженных книг, 0 задолженностей).
```
@startuml

actor Посетитель
actor Библиотекарь
participant "ИСБ" as System
participant "Учетная запись" as Account


  

Посетитель -> Библиотекарь : подать бумажную анкету
Библиотекарь -> System : registerAccount(fullName: String, phone: String, email: String)
activate System
activate Account
System -> Account : setStatus("Активен")
deactivate Account
Библиотекарь <- System : регистрация завершена
deactivate System
Посетитель <- Библиотекарь : регистрация завершена, читательский билет (qr-code)


@enduml
```
