Выдача книги читателю

Процесс:

1. Читатель предъявляет билет с qr-кодом
2. Библиотекарь считывает qr-code
3. Система проверяет статус читателя (нет ли штрафов, не превышен ли лимит)
4. Поиск книги в шкафу
5. Система фиксирует выдачу: какая книга, кому, и до какого числа нужно вернуть
6. Система меняет статус книги на "На руках".
```
@startuml

actor Читатель
actor Библиотекарь
participant "ИСБ" as System
participant "Учетная запись" as Account
participant "Каталог" as Catalog

Читатель -> Библиотекарь : предъявляет читательский билет

Библиотекарь -> System : сканирует qr-code
activate System

System -> Account : checkStatus(id): Boolean
activate Account
Account --> System : статус: активен
deactivate Account

System -> Catalog : searchBooks(title, author)
activate Catalog
Catalog --> System : экземпляры книги
deactivate Catalog

System -> Account : issueBook(accountId, bookId)
activate Account
Account --> System : книга выдана
deactivate Account


Библиотекарь <- System : подтверждение выдачи
deactivate System

Библиотекарь -> Читатель : выдает книги

@enduml
```
